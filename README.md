# ros_turtlesim

&emsp;Passo 1: Abra um terminal e inicialize o ROS com `roscore`.

&emsp;Passo 2: Em outro terminal, execute o turtlesim com `rosrun turtlesim turtlesim_node`.

&emsp;Passo 3: Em outro terminal, entre na pasta do projeto e rode o script com `source /opt/ros/noetic/setup.bash && cd /path/to/ros_turtlesim && python3 src/turtle/turtle/turtle.py`.
